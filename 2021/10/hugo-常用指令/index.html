[Hugo] 常用指令

2021年10月01日 11:29:08

使用hugo建立靜態網頁常用方法

建立新專案

hugo new site 專案名稱

新建文章

hugo new posts/文章名稱.md

啟動localhost（不包含草稿）

hugo server

啟動localhost（包含草稿）

hugo server -D

打包Hugo Site，可加入參數「-D」決定是否打包草稿。

hugo
